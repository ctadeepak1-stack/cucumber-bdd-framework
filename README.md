# 🚀 Cucumber BDD Selenium TestNG Framework

## 📌 Project Description
This project is an automation testing framework built using Selenium, Cucumber (BDD), and TestNG. It follows the Page Object Model (POM) design pattern and supports scalable and maintainable test automation.

---

## 🛠️ Tech Stack
- Java  
- Selenium WebDriver  
- Cucumber (BDD)  
- TestNG  
- Maven  
- Git & GitHub  

---

## ✨ Features
- BDD approach using Gherkin syntax  
- Page Object Model (POM) design  
- TestNG integration  
- Hooks for setup and teardown  
- Reusable step definitions  
- Parallel execution support  
- HTML and JSON reporting

3. Run using:
   - TestNG runner class  
   OR  
   - testng.xml
