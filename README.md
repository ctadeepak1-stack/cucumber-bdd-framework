# Cucumber BDD Selenium Framework

## Project Description
This project is an automation framework built using:
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Java

## Features
- Page Object Model (POM)
- Cucumber Feature Files
- Step Definitions
- TestNG Runner
- Maven Build

## Tools & Technologies
- Java
- Selenium
- Cucumber
- TestNG
- Maven

## How to Run
1. Clone the repository
2. Open in IntelliJ
3. Run TestRunner.java

## Project Structure
- src/test/java → Step Definitions & Runner
- src/main/java → Page Classes
- features → Feature files
